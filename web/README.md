# Estonia AI Kit - Web UI

Modern web interface for the Estonia AI Kit toolkit.

## Features

- 🎨 Modern, responsive UI built with React and TailwindCSS
- 🏰 Dashboard with statistics and service overview
- 📊 MCP Servers documentation and setup guides
- 💻 CLI Tools reference and usage examples
- 📚 Comprehensive documentation

## Tech Stack

- **React 18** - UI framework
- **TypeScript** - Type safety
- **Vite** - Build tool and dev server
- **TailwindCSS** - Styling
- **React Router** - Navigation
- **Lucide React** - Icons

## Getting Started

### Installation

```bash
cd web
npm install
```

### Development

```bash
npm run dev
```

The application will be available at `http://localhost:3000`

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
